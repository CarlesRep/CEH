## Cómo Usar este Repositorio

Este repositorio está organizado por módulos temáticos. Cada módulo contiene:
- Documentación detallada sobre conceptos
- Técnicas y metodologías
- Herramientas relacionadas
- Contramedidas y defensas

Se recomienda seguir los módulos en orden para una comprensión progresiva de los conceptos del CEH.
